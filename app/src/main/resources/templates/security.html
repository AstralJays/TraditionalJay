Story · Critical VM Compromise

Full chain

Runs all three steps with short gaps so detections can split cleanly. Log4Shell defaults to on-box LDAP (127.0.0.1:1389) — no laptop firewall ports required. Point C2 at a host the VM can reach.

  1. Leave LDAP + C2 at 127.0.0.1 — on-box attackers start at boot.
  2. Optional external C2: python3 tools/c2-listen.py --port 4444 on a reachable host.
  3. Optional laptop LDAP: ./tools/run-log4shell-ldap.sh --codebase-host YOUR_IP (needs inbound 1389+8000).
